Creating Variables with CSS !


Everything doesn't have to be a class!

Define your own CSS variables for easier interpretation of your code. Define variables at the :root level and your custom attributes that will accompany your variables. The variables are defined at the :root level. The attributes will hold the property names such as color, border, etc. The variables will be the property values! Set your property values as initial variables at the beginning of your document for easy updating. Set your custom attributes within your HTML code to make sure that your code is more easily understood.
